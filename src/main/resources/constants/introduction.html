Introduction

Les principales sources pour établir la descendance d'Ignace Douxchamps ont été les suivantes:

  • le livre, « La Famille Douxchamps », écrit par Hervé Douxchamps et publié en 1973 chez Heule.
  • le site internet d'Hervé Vander Heyden, liens : Ignace Douxchamps.
  • les dossiers des candidats boursiers.

Toutes ces données ont été encodées dans une base de données en utilisant le logiciel « Legacy », spécialement conçu pour l'établissement de généalogie.

Nous avons essayé d'être le plus complet possible en fonction des données en notre possession, mais il nous manque pas mal de renseignements, principalement dans les trois dernières générations.

Si dans votre branche, vous constatez des manquements ou des erreurs, nous vous demandons dès lors de nous contacter afin que nous puissions compléter la base de données.
